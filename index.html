Kitspace (formerly called Kitnic) is a registry of open hardware electronics projects that are ready for you to order and build. It could be described as a "Thingiverse for electronics". Click on any project to get further info, download the Gerbers and see the bill of materials.

To quickly purchase the parts from various retailers you should install the 1-click BOM extension. It's pretty useful on it's own too and can be used on other sites. Read more about it here.

Help make an open hardware repository of useful electronics projects! Submit your own project to have it listed here. Follow Kit Space on Twitter, Facebook, Reddit, Hackaday.io or GitHub to get updates as we progress and add content.
